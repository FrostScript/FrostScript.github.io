Hexo 使用完整指南

Hexo 是一个快速、简洁且高效的博客框架，它基于 Node.js 开发，能够帮助你轻松地创建个人博客网站。本文将带你从零开始搭建和使用 Hexo 博客。

七、常用命令简写

命令 简写 说明
hexo generate hexo g 生成静态文件
hexo server hexo s 启动服务
hexo deploy hexo d 部署
hexo clean 清除缓存
hexo new hexo n 新建文章
hexo help hexo h 帮助
hexo version hexo v 版本信息
