Hexo 博客搭建详细教程

1. 环境准备

1.1 安装 Node.js

  • 访问 Node.js 官网
  • 下载并安装 LTS（长期支持）版本
  • 安装完成后，打开命令行输入 node -v 和 npm -v 验证安装

1.2 安装 Git

  • 访问 Git 官网
  • 下载并安装 Git
  • 安装完成后，打开命令行输入 git --version 验证安装

8. 常用命令

  • hexo new post "文章名" - 新建文章
  • hexo new page "页面名" - 新建页面
  • hexo clean - 清除缓存
  • hexo generate - 生成静态文件
  • hexo server - 启动服务器
  • hexo deploy - 部署网站

9. 注意事项

  1. 每次修改配置后需要重启服务器
  2. 部署前建议先在本地预览
  3. 记得定期备份你的博客内容
  4. 可以使用 hexo g -d 命令一键生成并部署
