# 个人静态博客网站

这是一个使用HTML5、CSS3和JavaScript构建的现代化个人静态博客网站。

## 特点

- 响应式设计，适配各种设备
- 现代化的UI界面
- 平滑的动画效果
- 简洁的文章展示
- 社交媒体集成

## 项目结构

```
.
├── index.html          # 主页
├── styles/
│   └── main.css       # 主样式文件
├── js/
│   └── main.js        # 主JavaScript文件
└── README.md          # 项目说明文件
```

## 使用方法

1. 克隆仓库到本地
2. 使用任意静态文件服务器运行项目
3. 根据需要修改内容和样式

## 自定义

- 修改 `index.html` 中的内容
- 在 `styles/main.css` 中调整样式
- 在 `js/main.js` 中添加新的交互功能

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome 图标
